En este repositorio se van a proporcionar diferentes códigos fuente para la utilización 
de la plataforma multi computacional MATLAB y SIMULINK. Muchos de los contenidos de este 
repositorio, facilitarán al usuario, la creación de audio benches para analizar señales 
de audio, utilizando para ello, la librería para digital signal processing (DSP).

Creación de filtros de primera respuesta (FIR), controles vía MIDI, e información relevante 
acerca de los procesos de conversión de dicho tipo de señales. El usuario que no está 
familiarizado con el tipo de comandos o consultas que se pueden llegar a ejecutar a través 
de MATLAB, van a poder encontrar, una guia que les facilitara la tarea de familiarización 
con esta aplicación, pensada para todo tipo de proyectos.

(Los códigos fuente van a ser proporcionados a lo largo de esta semana)
